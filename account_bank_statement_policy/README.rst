=============================
Account Bank Statement Policy
=============================

This module adds "Bank Statement Policy" and
"Cash Register Policy" to Journal
    
Installation
============

To install this module, you need to:

1.  Clone the branch 9.0 of the repository https://github.com/open-synergy/opnsynid-account-financial_tools
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Apps -> Apps*
5.  Search For *Account Bank Statement Policy*
6.  Install the module
